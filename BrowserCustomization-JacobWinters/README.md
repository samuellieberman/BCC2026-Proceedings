# Browser customization: beyond the basics

## Jacob Winters

### Boise Code Camp 2026 - May 2nd, 2026

Learn why uBlock Origin calls itself a "wide-spectrum content blocker", where to find the three accessibility settings graphic designers don't want you to know about, and how to bring Tor experts to your door with torches and pitchforks. We're also going to turn a search engine home page into an implementation of the classic `xeyes` utility, because we can.
