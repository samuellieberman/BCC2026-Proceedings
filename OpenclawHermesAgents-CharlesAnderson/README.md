# From OpenClaw to Hermes: Running Local Autonomous Agents in 2026

## Charles "Chuck" Anderson

### Boise Code Camp 2026 - May 2nd, 2026

Want AI that actually does things for you, without sending your data to the cloud? In this hands-on session, we'll explore how to build and run powerful personal AI agents locally using OpenClaw and the exciting new Hermes Agent from Nous Research.

You'll learn how to set up autonomous agents that handle real tasks like email, calendar, research, automation, and more.

Whether you're a developer curious about local LLMs, agents, or persistent memory systems, this talk will give you practical steps to run private, customizable agents on your own hardware or a cheap VPS. Leave with working examples and the knowledge to start building agents that grow with you.

No prior agent experience required, just bring your curiosity!
