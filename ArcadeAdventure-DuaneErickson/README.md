# Code Your Own Arcade Adventure

## Duane Erickson

### Boise Code Camp 2026 - May 2nd, 2026

Code your own retro game! In this hands-on session for ages 8 and older, families will jump into Microsoft MakeCode Arcade together, using block coding to design and build their own games inspired by a classic top down adventure game. Kids and parents work together to remix and bring their creations to life. No experience needed.
