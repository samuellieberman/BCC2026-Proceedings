# It's Dangerous to Go Alone

## Aaron Bynum

### Boise Code Camp 2026 - May 2nd, 2026

Soft Skills, Networking, and the Current State of the Games Industry

This is a redux of a presentation I recently gave to the BSU GIMM program and provides an inside look at the current state of the games industry, provides tips, tricks, and tools to help students and aspiring game developers of all ages get a head start on getting the coveted game dev position they've been hoping for.

I will also give a brief look at a personal game development project I've been working on in Unreal Engine 5.
