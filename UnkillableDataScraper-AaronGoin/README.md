# The Unkillable Scraper: Data Collection That Doesn't Quit

## Aaron Goin

### Boise Code Camp 2026 - May 2nd, 2026

Building a web scraper that works once is easy. Building one that keeps working when sites update, deploy anti-bot measures, or change their structure overnight? That's the real challenge.

This session covers practical data collection through real examples: 
- Scraping unique stats for fantasy football
- Pulling business data from sites that don't offer APIs or easy exports 
- Using techniques for detection evasion
- Designing methods that automatically adapt to website changes

You'll walk away with techniques that work in real situations. This isn't about theory or perfect setups. It's about getting data when you need it and making sure your scraper doesn't die the next day when the site changes something. If you've ever needed data that wasn't easy to get, or you're tired of fixing the same scraper over and over, this session will help.
