# Introduction to Kotlin and Jetpack Compose

## Doug Reece

### Boise Code Camp 2026 - May 2nd, 2026

Kotlin with Jetpack Compose is the modern way to build Android apps. Using a declarative, Kotlin-first approach, it’s faster to write, easier to reason about, and far less boilerplate-heavy than XML-based layouts. Compose lets you build reactive UIs that automatically update with state changes, improve code readability, and align closely with modern development patterns used across platforms. 

In this session we’ll cover: 
  What Jetpack Compose is and how it differs from XML layouts 
  Declarative UI concepts 
  Basic composables (Text, Button, Column, Row) 
  Layouts, modifiers, and styling 
  Handling user input and events 
  Previewing composables in Android Studio 
  And other things...
