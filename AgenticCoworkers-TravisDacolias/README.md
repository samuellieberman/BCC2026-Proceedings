# Beyond the Prompt: Building Digital Coworkers with Agentic Workflows

## Travis Dacolias

### Boise Code Camp 2026 - May 2nd, 2026

AI agents are transforming how we use large language models, moving us from simply asking questions to delegating complex tasks to digital coworkers. But building a real-world AI agent requires much more than a good prompt: it needs memory, tool access, multi-step reasoning, and the ability to operate within clear boundaries. In this session, we’ll move beyond basic LLM usage and break down how agentic workflows are actually designed.

We’ll start with the core architecture behind practical AI agents and the kinds of problems they are good at solving, especially repetitive, structured, low-risk workflows where human review can still be layered in. From there, we’ll look at several common agent design patterns and discuss when each pattern is useful, where it fails, and how to combine them into reliable systems.

The session will then shift into a visual walkthrough of a working agentic workflow that coordinates LLM reasoning, external tools, and control logic to complete a multi-step task. Rather than focusing on heavy coding, the emphasis will be on orchestration, state, decision points, and "human-in-the-loop" checkpoints so attendees can understand how to design agents that are both useful and trustworthy.

Attendees will leave with a practical mental model for evaluating agent opportunities, choosing the right design pattern, and translating a one-shot LLM interaction into a workflow that can reason, act, and improve over multiple steps.
