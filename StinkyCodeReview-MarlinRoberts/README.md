# Stinky Code - A Review of Common Code Smells

## Marlin Roberts

### Boise Code Camp 2026 - May 2nd, 2026

This presentation introduces developers to the concept of "code smells" — common patterns in code that signal poor design, inefficiency, or future maintenance problems. Designed for a mixed-experience audience. These are best documented in Clean Code by Robert C Martin, but with some additions.

The presentation covers 12 named code smells including God Objects, Long Methods, Duplicate Code, Magic Numbers, Feature Envy, Dead Code, Long Parameter Lists, Primitive Obsession, Shotgun Surgery, Bad Comments, Data Clumps, and Switch Statements. Each smell is presented with a description, an explanation of why it causes problems, and concrete before/after code examples showing how to fix it.

A bonus "Gotchas" section covers six common Python-specific traps that frequently catch beginners off guard.
